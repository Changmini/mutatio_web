# mutatio web project

논문품질측정 및 논문 정보 공유 사이트

논문 품질 / 관심 키워드 추천 / 논문 추천 / 최근 이슈 논문 / 차트 정보 등을 제공하여
논문 이용자가 여러 검색엔진을 돌아다니며, 논문을 수집하고 비교하는 시간을 줄일 수 있도록 한다.
